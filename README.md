# YouTube Downloader

Aplicação desenvolvida em Python para baixar vídeos do YouTube utilizando uma interface gráfica construída com CustomTkinter.

## Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

## Execução

Execute o arquivo principal:

```bash
python main.py
```

## Como utilizar

1. Cole a URL do vídeo do YouTube no campo indicado.
2. Escolha a pasta onde o vídeo será salvo.
3. Clique em "Baixar Vídeo".
4. Aguarde a conclusão do download.

## Tecnologias utilizadas

* Python
* CustomTkinter
* Pytubefix

## Observações

* É necessário ter conexão com a internet para realizar os downloads.
* Dependendo das alterações realizadas pelo YouTube, pode ser necessário atualizar as dependências do projeto.
